# 配置管理规范（基于麒麟项目）

## 配置分类
1. **服务器配置** - 端口、主机、CORS
2. **数据库配置** - 连接、连接池、时区
3. **认证配置** - JWT、密码、会话
4. **租户配置** - 多租户设置
5. **业务配置** - 分页、上传、缓存
6. **安全配置** - 速率限制、CORS、Helmet
7. **监控配置** - 日志、性能

## 环境变量命名
- 使用下划线分隔: `DB_HOST`, `JWT_SECRET`
- 分类前缀: `SERVER_PORT`, `DB_CONNECTION`
- 布尔值: `ENABLE_LOGGING=true`

## 配置验证
- 启动时自动验证必需配置
- 提供默认值仅用于开发
- 生产环境必须显式配置
