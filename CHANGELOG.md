# 麒麟云点餐 — 版本发布记录

## v0.4.0 (2026-04-27) 套餐体系 & 品牌落地页

### 新增功能
- **首页重写为品牌推广落地页** — Hero 区 + 功能展示 + 数据条 + Mockup + 定价
- **首页麒麟 SVG 图标** — 导航栏/Hero/Footer 统一品牌标识
- **套餐限制系统** — 定义 FREE/SINGLE/PRO/ENTERPRISE 可见
- **注册自动创建租户** — 注册时自动创建 Tenant（plan=FREE, trialEndsAt=14天后）
- **新增单店版套餐（¥38/月）** — 亲民定价，1门店/2打印机/90天数据留存
- **注册页套餐选择** — 四套卡片式套餐选择 + 底部功能对比表
- **注册页 localStorage 草稿** — 跳转后自动恢复表单内容
- **服务条款页面** — 完整10章法律条款
- **隐私政策页面** — 完整9章隐私说明
- **店招卡片化** — 扫码页店铺背景左右留白 + 圆角

### 优化改进
- 图形验证码暗纹/干扰线优化，数字加大
- 四个登录页统一加图形验证码
- 登录标识改为"账号"（支持邮箱/用户名/手机号）
- 所有登录页加"记住账号"功能
- 店长端首页店铺卡片加数据看板

### 技术架构
- 后端：Fastify + Prisma + PostgreSQL
- 前端：React + TypeScript + Vite
- 生产：Nginx HTTS + PM2 cluster 模式
- 部署：saas.openyun.xin
